# SmartCart Campus — Group Buying App

A React + Vite web app mockup for campus group-buying.

## Quick Start (Development)

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

The built files will be in the `dist/` folder — upload them to any static host (Netlify, Vercel, GitHub Pages, cPanel, etc.).

## Hosting

After running `npm run build`, deploy the `dist/` folder:

- **Netlify**: Drag & drop the `dist/` folder at netlify.com/drop  
- **Vercel**: `npx vercel --prod` from this folder  
- **GitHub Pages**: Push `dist/` contents to your `gh-pages` branch  
- **cPanel / shared hosting**: Upload `dist/` contents to your `public_html` folder
